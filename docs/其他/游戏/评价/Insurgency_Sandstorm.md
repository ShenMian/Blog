叛乱: 沙漠风暴是一款偏硬核的第一人称设计游戏.  

偏硬核的元素主要有以下几点:  

- 平均 TTK 短.
- 真实的弹药管理: 备弹为携带的弹匣, 且只能查看单个弹匣内的大致子弹数.
- HUD 有限: 比如没有手雷指示器, 需要根据手雷落地碰撞的响声判断手雷的落点.

## 优点

- 枪械模型质量很高.
- PvE + PvP.
