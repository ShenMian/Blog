# 面向对象编程

**英文**: Object-Oriented Programming.  

## 三个特征

- 封装: 提高了类的易用性.
- 继承: 提高了代码的复用率.
- 多态: 提高了程序的可拓展性.

不同编程语言实现三个特征的方式不同, 比如:  

- C++ 和 Java 是基于类(Class)的面向对象.
- JavaScript 是基于原型(Prototype)的面向对象.
- Lua 是基于原表(Meta Table)的面向对象.

## 优点

- 易于维护.
- 可重用性.
- 可拓展性.

## 缺点

- 不太适用于嵌入式开发.
- 性能略低于面向过程编程.
