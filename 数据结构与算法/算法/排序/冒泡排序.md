# Bubble Sort

冒泡排序.

## 指标
**时间复杂度**: O(n^2)
**空间复杂度**: O(1)
**稳定性**: 稳定

## 算法
对比 data[i] 和 data[i + 1] 的大小, 若顺序与排序要求的相反则交换这两个值. 从头到尾对比一遍后可以确保最后一个元素的位置是正确的.

## 实现
```cpp
void sort(std::vector<int>& data) // 升序排序
{
  for(size_t i = 0; i + 1 < data.size(); i++)
    for(size_t j = i + 1; j < data.size(); j++)
      if(data[i] > data[j])
        std::swap(data[i], data[j]);
}
```
